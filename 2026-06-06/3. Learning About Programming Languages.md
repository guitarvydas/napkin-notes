#### ASTs 
- BNF and eBNF
	- BNF is just a DSL for writing grammars
	- grammars are transformed into data structures called ASTs (Abstract Syntax Trees)
	- when source code is compiled it is turned into a data structure which is a pruned AST, called a CST (Concrete Syntax Tree)
	- AST specifies everything that is allowed, CST represents only what is in the source code being compiled
