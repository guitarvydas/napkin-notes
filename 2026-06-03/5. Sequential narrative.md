I'm not sure that this is the case. Sequentialism is wishful thinking and reality denial. Sequentialism is used by mathematicians, physicists and software programmers. Hardware is designed with async-by-default components, designing-in sync only on demand. 

Humanity has developed protocols for dealing with async - handshaking, waiting & not-waiting for meetings to begin. Software used to be orders of magnitude smaller before "C" popularized the function-based, sequential approach, yet we created things like Sketchpad, Visicalc, etc. 

Nobel laureate Ilya Prigogene is on record saying that this approach has set physics back by 100 years.
