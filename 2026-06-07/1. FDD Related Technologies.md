The techniques in include
- PBP - Parts Based Programming
- T2T - text to text transmogrification (convert a custom syntax to runnable code in an existing language)
- existing command line tools (awk, sed, jq, etc.)
- Prolog for exhaustive search
- Ceptre
- @make
	- how to stop adding bandaids to 'make' and do something simpler that allows for building and using tools that consist of combinations of the above technologies. 
	- For example, if you build a state-machine diagram to Python transmogrifier that operates like an enhanced pipeline using many tools, you should be able to use it to draw little state machines in other projects
